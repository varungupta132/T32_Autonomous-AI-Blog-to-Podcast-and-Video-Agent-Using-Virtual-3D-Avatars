## 📝 Example Workflows

### Workflow 1: Text → Podcast → Animation
```
1. Open: http://localhost:8080
2. Paste text: "Host: Hello! Guest: Hi there!"
3. Click "Generate Podcast"
4. Download audio
5. Open: http://localhost:5003
6. Upload audio
7. Click Play
8. Watch animated avatars!
```

### Workflow 2: Text → Professional Video
```
1. Open: http://localhost:8080
2. Paste text
3. Generate podcast
4. Download audio
5. Open: http://localhost:5004
6. Upload audio
7. Choose avatar (Emily, Marcus, etc.)

9. Wait 2-3 minutes
```

## 🐛 Common Issues

### Port Already in Use?
```bash
# Stop running servers
taskkill /F /IM python.exe

# Or restart computer
```

### Avatars Not Showing?
- Refresh browser (Ctrl+R)
- Clear cache (Ctrl+Shift+R)
- Check console (F12)

### FAL API Error?
- Check internet connection
- Verify API key in .env
- Check FAL credits
